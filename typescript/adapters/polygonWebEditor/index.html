Hitbox Editor

Comandos

  • 🖱️ Esq: Adicionar Ponto / Selecionar
  • 🖱️ Dir: Desfazer / Apagar
  • ⌨️ Enter: Finalizar Polígono
  • ⌨️ Esc: Cancelar
  • 🖱️ Scroll: Mover Verticalmente
  • Shift + Scroll: Mover Horizontalmente
  • Ctrl + Scroll: Zoom (Focado no Mouse)
  • ⌨️ Shift + Delete: Limpar Todas (Rápido)
  • ⌨️ Ctrl + Z: Desfazer (Undo)
  • ⌨️ Ctrl + Y: Refazer (Redo)
